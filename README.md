# Tetris

A Classic Tetris (NES Tetris) clone written in bevy/rust.

## Features

- Game Play
  - [x] DAS (16 ticks / 6 ticks)
  - [ ] Glitched Color Palettes
  - [x] Level 39 Super Kill Screen
  - [x] Statistics

- Platform
  - [x] High FPS Support
  - [x] Cross-platform
